# assets/icons — iconos de la app

Pon aquí los PNG (recortados de tus packs), un archivo por icono, con estos nombres.
Recomendado: **PNG cuadrado, fondo transparente, 128×128 px** (o 256×256 para pantallas retina).

Cuando ya estén los archivos, en el código pon:
    window.ICONOS_PNG = true;
y la app usará las imágenes en vez de los emojis (con respaldo automático al emoji si falta alguno).
